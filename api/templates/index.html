ScanT3r - API

http://localhost:6040/scan/{SCANNER_ID}?url={TAGET}

example :
$ curl 'http://localhost:6040/scan/1?url=http://testphp.vulnweb.com/search.php?test=query%26searchFor=1%26goButton=go'
ID Scanner
1 XSS
2 SQLI
3 RCE
4 SSTI
5 CRLF
